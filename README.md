# 🦎 LizardWWG (Where We Goin')

Tiny QoL addon for Synastria.

Right-click a warp to instantly open the map for that zone so you can see where you're going or pick a spot for **attuned warps**.

---

## ✨ What It Does

- 🗺️ **Right-click warp → open map**
- 🎮 Optional keybind if you prefer hovering + pressing a key
- 🌍 Works for EK / Kalimdor / Outland / Northrend
- 🪶 Lightweight, no config needed

---

## 🕹️ Usage

When the warp window is open:
Left click → teleport
Right click → open map

Optional Keybind:
Esc → Key Bindings → LizardWWG

Hover a warp and press your key.

---

## 📦 Installation

1. Download the latest release  
2. Extract into: Interface/AddOns/
3. Folder should look like: Interface/AddOns/LizardWWG/
4. Restart the game
